Q1) Explain the purpose of using the var() function in css.
Ans) a) Reusability: CSS variables allow us to store values that we can reuse across our stylesheet. The var()
function lets us reference these values wherever needed, reducing redundancy and making our code
cleaner.

b) Global Changes: If we need to make a design change, we can update the variable's value once, and the
change will be applied to all instances using that variable.
